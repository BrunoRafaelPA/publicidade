# Sistema de Publicidade

Sistema completo de gerenciamento de publicidades com backend Rails e frontend Angular.

## Tecnologias

- **Backend:** Ruby on Rails 8.0.2
- **Frontend:** Angular 19.2.0
- **Banco:** PostgreSQL 16
- **Containerização:** Docker & Docker Compose

##  Pré-requisitos

- Docker
- Docker Compose
- Git
